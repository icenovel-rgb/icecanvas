# ICE Canvas

ICE Canvas is a Tauri desktop app for organizing personal notes, links, files,
videos, PDFs, and image cards on a canvas.

## Install

Use the MSI installer in the project root:

```text
ICE Canvas_0.1.6_x64.msi
```

The installer is generated by Tauri. It does not use VBS, hidden PowerShell, or
a background local HTTP server.

## Development

Requirements:

- Node.js
- Rust toolchain
- Visual Studio Build Tools with C++ tools

Build:

```powershell
$env:PATH="$env:USERPROFILE\.cargo\bin;$env:PATH"
npm.cmd run build
```

The generated MSI is created under:

```text
src-tauri\target\release\bundle\msi
```

## Notes

- App UI files live in `ui/`.
- Tauri backend files live in `src-tauri/`.
- Save overwrites the current `.canvas` file after the first Save As/open path.
- Save As always asks for a new path.
